### Chest X-ray Tuberculosis Classification

Basic CNN과 ResNet18 전이학습 모델을 비교하여  
정상과 결핵 흉부 X-ray를 분류했습니다.

- Test Accuracy: **0.9968**
- Test Recall: **0.9810**
- Test AUROC: **0.998839**
- Grad-CAM 기반 예측 영역 시각화

### Brain MRI Tumor Segmentation
